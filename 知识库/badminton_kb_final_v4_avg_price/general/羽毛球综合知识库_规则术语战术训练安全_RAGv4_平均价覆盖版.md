# 羽毛球术语词典

## 0. 2026-07-09 规则版本更新：3×15 过渡说明

> RAG 优先级：当用户问“现在规则是多少分”“比赛几分制”“20 平/14 平怎么办”时，先召回本节，再召回原有计分规则。

- **截至 2026-07-09**：多数通用解释仍需要以赛事当时采用的 BWF Laws / 赛事规程为准；原文中的 21 分制内容可作为“3×21 赛事/过渡期/历史规则”解释。
- **已批准的变更**：BWF 成员已批准采用 **3×15 计分系统**，官方新闻说明该系统将于 **2027-01-04** 生效。
- **3×15 结构**：三局两胜、每局先到 15 分；BWF 早前提案表述为 **3×15 with setting to 21**，即 14 平后通常需要领先 2 分，最高封顶至 21 分；最终执行细节以 2027 生效版 Laws 和赛事规程为准。
- **知识库回答策略**：2027-01-04 前不要直接把所有比赛说成 15 分制；2027-01-04 后不要继续把 21 分制当默认规则。若用户问历史比赛、业余约球或地方赛事，应提示“以该赛事/球馆约定为准”。


## 1. 比赛与计分术语

| 术语 | 英文 | 解释 | RAG 使用建议 |
|---|---|---|---|
| 回合 | rally | 从发球被击出开始，到球成死球为止的一段连续对抗。 | 用于解释“每球得分”“失误得分”。 |
| 一局 | game | 羽毛球比赛的组成单位。BWF 标准规则中，一场比赛通常采用三局两胜。 | 检索计分规则时优先召回。 |
| 一场比赛 | match | 由若干局组成的完整比赛。 | 回答“赢几局算赢”时使用。 |
| 每球得分 | rally point scoring | 每个回合结束后，赢得该回合的一方得 1 分。 | 回答新手计分问题。 |
| 局点 | game point | 一方再赢 1 分即可赢下该局。 | 属于常用比赛语言，不是单独判罚规则。 |
| 赛点 | match point | 一方再赢 1 分即可赢下整场比赛。 | 用于赛事解说类问答。 |
| 平分 | deuce / tie | 双方分数相同。3×21 规则中 20:20 后通常需要领先 2 分，29:29 时先到 30 分者胜；3×15 规则已批准 2027-01-04 生效，14 平后的 setting 细节以生效版 Laws/赛事规程为准。 | 回答“20 平怎么办”。 |

## 2. 场地区域术语

| 术语 | 英文 | 解释 | 关键点 |
|---|---|---|---|
| 单打边线 | singles sideline | 单打有效宽度的边界线。 | 单打比双打窄。 |
| 双打边线 | doubles sideline | 双打有效宽度的外侧边界线。 | 双打比单打宽。 |
| 端线/底线 | back boundary line | 场地后边界线。 | 单打和双打后场边界相同，但双打发球有专门长发球线。 |
| 短发球线 | short service line | 发球必须越过的前方界线。 | 发球落点没有到短发球线之外为发球失误。 |
| 双打长发球线 | long service line for doubles | 双打发球区后界线，位于端线前。 | 只限制双打发球落点，不等于双打回合后场边界。 |
| 中线 | centre line | 将左右发球区分开的线。 | 发球站位和落点判断使用。 |
| 前场 | forecourt | 靠近网前的区域。 | 对应搓、放、推、扑、勾等技术。 |
| 中场 | midcourt | 前后场之间区域。 | 对应平抽、挡、接杀、过渡。 |
| 后场 | rearcourt | 靠近底线的区域。 | 对应高远、吊、杀、劈吊等技术。 |

## 3. 基本技术术语

| 术语 | 英文 | 解释 | 常见问题 |
|---|---|---|---|
| 发球 | serve | 每个回合的开始击球。 | 发球高度、站位、脚不能踩线等规则容易出错。 |
| 接发球 | return of serve | 接发方对发球的第一拍处理。 | 双打战术中价值很高。 |
| 高远球 | clear | 将球打到对方后场高远区域。 | 新手后场不到位常见。 |
| 平高球 | attacking clear / drive clear | 弧线较低、速度较快的后场球。 | 风险高于普通高远球。 |
| 吊球 | drop shot | 从后场把球轻放到对方前场。 | 需要和高远、杀球动作一致性配合。 |
| 杀球 | smash | 向下快速进攻击球。 | 不是所有场景都适合强杀。 |
| 点杀 | stick smash / half smash | 力量较小但速度快、落点突变的杀球。 | 常用于连贯和抓空当。 |
| 劈吊 | sliced drop | 用切削制造变线或减速的吊球。 | 对手感要求高。 |
| 放网 | net shot | 在网前把球轻放过网。 | 质量看高度、旋转、贴网程度。 |
| 搓球 | spinning net shot | 网前利用摩擦制造旋转。 | 风险高，容易不过网或过高。 |
| 勾对角 | cross net shot | 网前将球勾到对方对角网前。 | 对击球点和手感要求高。 |
| 推球 | push | 中前场把球快速推向空当或后场。 | 双打接发常用。 |
| 扑球 | net kill | 在网前对高过网的球快速下压。 | 注意不能触网或过网击球。 |
| 平抽挡 | drive and block | 中场快速平行来回击球、防守挡网结合。 | 双打高频技术。 |
| 接杀 | smash defense | 防守对方杀球。 | 包括挡网、抽对角、挑后场。 |

## 4. 战术术语

| 术语 | 解释 | 适用场景 |
|---|---|---|
| 拉吊突击 | 通过高远、吊球调动对手，再抓浅球突击。 | 单打常见。 |
| 控后场 | 通过高远、平高、挑球压制对方后场。 | 对手后退慢或反手弱。 |
| 压反手 | 持续把球打向对手反手区域。 | 单打常用策略。 |
| 杀上网 | 后场杀球后快速跟进前场封网。 | 单打、双打均可出现。 |
| 前后站位 | 双打进攻站位，前场队员封网，后场队员进攻。 | 双打进攻。 |
| 左右站位 | 双打防守站位，二人横向分区防守。 | 接杀、防守。 |
| 轮转 | 双打中根据球路和攻防转换调整前后/左右位置。 | 双打核心。 |
| 发接发前三拍 | 发球、接发、第三拍的连续争夺。 | 双打和混双高价值环节。 |

## 5. 装备术语

| 术语 | 解释 | 注意 |
|---|---|---|
| 3U / 4U / 5U | 球拍重量规格区间，不同品牌标注可能略有差异。 | 不能只看 U 数，还要看平衡和中杆。 |
| G5 / G6 | 手柄粗细规格。 | 品牌体系可能不同。 |
| 平衡点 | 球拍重心位置。 | 可影响头重、均衡、头轻手感。 |
| 中杆硬度 | 球拍杆身弯曲难易程度。 | 硬杆要求发力更完整。 |
| 磅数 | 球线张力。 | 高磅通常更要求发力和击球稳定性。 |
| 甜区 | 拍面中较容易打出有效回球的区域。 | 甜区大通常更宽容。 |

## 6. 不建议直接作为事实的词

以下词多来自玩家口碑或商家描述，建议作为“主观感受标签”，不要作为硬事实：

- 暴力
- 秒杀
- 神器
- 天花板
- 超级适合所有人
- 职业同款所以一定好用
- 新手闭眼入

RAG 处理建议：当用户问这类词时，模型应转化为可解释维度，例如“头重程度、重量规格、中杆硬度、挥速、容错、适合水平”。

## 1. 资料依据与版本边界

本文件依据 BWF Statutes 中的 Laws of Badminton 以及 BWF 简化规则资料整理。BWF 官方 Statutes 页面显示，规则文件会按章节维护，并且存在生效日期版本。因此，本知识库用于解释通用规则；正式赛事判罚必须以赛事采用的最新 BWF 文件和赛事规程为准。

## 2. 比赛基本形式

羽毛球常见项目包括：

| 项目 | 场上人数 | 说明 |
|---|---:|---|
| 男子单打 | 1 对 1 | 使用单打边线。 |
| 女子单打 | 1 对 1 | 使用单打边线。 |
| 男子双打 | 2 对 2 | 使用双打边线。 |
| 女子双打 | 2 对 2 | 使用双打边线。 |
| 混合双打 | 2 对 2 | 每方一男一女，场地边线同双打。 |

## 3. 计分规则

BWF Laws 中的通用比赛结构为三局两胜，除非另有安排。标准 21 分制下：

- 每个回合胜方得 1 分。
- 一局通常先到 21 分者胜。
- 若 20:20，需领先 2 分。
- 若 29:29，先到 30 分者胜。
- 一场比赛通常为三局两胜。

RAG 问答注意：如果用户问“现在是不是 15 分制”，应说明 BWF 曾测试/使用过替代计分方案或特定赛事安排，但通用解释应回到当前赛事采用规则，不能默认所有比赛都一样。

## 4. 发球与接发基本规则

发球的核心要求：

| 规则点 | 解释 |
|---|---|
| 对角发球 | 发球必须从本方发球区打到对方斜对角发球区。 |
| 站位不能踩线 | 发球员和接发球员应站在相应发球区内，脚不得触及界线。 |
| 双脚接触地面 | 发球动作开始到发出前，发球员和接发球员的双脚应与地面保持接触。 |
| 连续向前挥拍 | 发球动作应连续向前，不能明显停顿欺骗。 |
| 发球高度 | BWF 现代规则采用固定高度服务规则，具体以最新 Laws 条文为准。 |
| 发球落点 | 必须落入对方斜对角发球区，且越过短发球线。 |

注意：发球高度、裁判设备、发球裁判执行口径与赛事级别有关，知识库应避免用“业余场一定按某个高度精确判罚”这类表达。

## 5. 单打发球站位

| 发球方分数 | 发球位置 | 接发位置 |
|---:|---|---|
| 偶数分 | 右发球区 | 对方右发球区 |
| 奇数分 | 左发球区 | 对方左发球区 |

单打中，发球方赢得回合后继续发球，并根据自己的分数奇偶换发球区。

## 6. 双打发球站位

双打同样按照发球方分数奇偶决定发球区：

| 发球方分数 | 发球区 |
|---:|---|
| 偶数分 | 右发球区 |
| 奇数分 | 左发球区 |

双打中，每方只有一次发球权。发球方得分后，原发球员继续发球，并与搭档交换左右发球区。接发方得分后，获得下一回合发球权。

## 7. 界内与界外

| 场景 | 判断 |
|---|---|
| 球落在线上 | 通常视为界内。 |
| 单打回合 | 单打边线以内、底线以内为界内。 |
| 双打回合 | 双打边线以内、底线以内为界内。 |
| 双打发球 | 宽度按双打边线，长度按双打长发球线。 |
| 单打发球 | 宽度按单打边线，长度到底线。 |

## 8. 常见违例/失误

| 类型 | 解释 |
|---|---|
| 发球失误 | 发球没有进入正确发球区、未越过短发球线、发球动作违规等。 |
| 出界 | 球落在有效区域外。 |
| 下网 | 球没有越过球网。 |
| 触网 | 比赛进行中，运动员或球拍触及球网或网柱，通常构成违例。 |
| 过网击球 | 击球点在对方场区一侧，通常构成违例；但随挥过网需按规则具体判断。 |
| 连击 | 一方连续击球两次，或同一击球过程中出现规则禁止的连续触球。 |
| 妨碍对手 | 阻挡对方合法击球、故意干扰等。 |

## 9. RAG 推荐回答结构

用户问规则时，建议回答：

1. 先给结论；
2. 说明适用场景：单打/双打/发球/回合中；
3. 说明边界：正式比赛以 BWF 与赛事规程为准；
4. 给 1 个简单例子；
5. 如果涉及争议判罚，提示“需要裁判根据现场事实判断”。

## 10. 低置信度/需核验项

以下内容可能随规则版本、赛事规程、技术官员指引变化，生成回答时需提示核验：

- 固定发球高度执行细节；
- 电子发球判罚设备要求；
- 特定赛事是否采用替代计分；
- 裁判临场对延误、行为不端的处理流程。

# 用户水平分级标准

## 1. 说明

BWF Shuttle Time 和教练教育资料用于学校/入门教学与教练培训，但并不直接给出面向电商推荐的“新手/中级/高级”统一商业分级。因此本文将公开教学资料中的技术、移动、战术和训练要素整理成 RAG 可用的分级标签。该分级是“知识库标签体系”，不是官方运动员等级认证。

## 2. 分级总览

| 等级 | 标签 | 核心判断 | 推荐用途 |
|---|---|---|---|
| L0 | 完全新手 | 不熟悉规则、握拍、发球和基本回合 | 入门规则、基础装备 |
| L1 | 初学者 | 能发球、简单对打，但动作不稳定 | 新手选拍、基础训练 |
| L2 | 入门进阶 | 能完成高远、吊、挑、网前等基本球路 | 选拍细分、步法训练 |
| L3 | 中级 | 有单双打意识，能形成基本战术 | 战术指导、型号对比 |
| L4 | 中高级 | 动作较稳定，能根据对手调整打法 | 高端器材、专项战术 |
| L5 | 高级/竞赛 | 有比赛经验，能控制节奏和轮转 | 精细对比、比赛复盘 |

## 3. 技术维度

| 技术项 | L0-L1 表现 | L2-L3 表现 | L4-L5 表现 |
|---|---|---|---|
| 握拍 | 正反手转换不清 | 能区分正反手和网前握拍 | 能在高速对抗中自然切换 |
| 发球 | 经常不过网/出界 | 能稳定发短球和高远球 | 能通过落点、节奏制造优势 |
| 后场球 | 难以打到底线 | 能完成基本高远、吊、杀 | 动作一致性好，线路变化多 |
| 网前 | 容易挑高或下网 | 能放网、推球、挑球 | 能搓、勾、扑并控制高度 |
| 防守 | 接杀被动，回球单一 | 能挡网或挑后场 | 能防守转反击 |
| 步法 | 跑着追球 | 有启动、回中意识 | 步点精准，重心转换稳定 |

## 4. 战术维度

| 等级 | 单打战术表现 | 双打战术表现 |
|---|---|---|
| L0-L1 | 主要把球打过网 | 容易两人抢球或漏中路 |
| L2 | 知道拉开前后左右 | 初步知道前后/左右站位 |
| L3 | 能使用拉吊突击、压反手 | 知道轮转、接发前三拍重要 |
| L4 | 能针对对手弱点组织球路 | 能按攻防转换主动补位 |
| L5 | 能做比赛计划和实时调整 | 有稳定分工、轮转和封网意识 |

## 5. 体能与伤病维度

| 等级 | 身体能力特征 | RAG 建议 |
|---|---|---|
| L0-L1 | 易疲劳，动作代偿明显 | 选择轻量、容错高装备；训练先控制强度 |
| L2-L3 | 可完成常规训练，但专项力量不足 | 加入步法、多球、核心和下肢稳定 |
| L4-L5 | 可承受较高强度专项 | 注意周期安排、伤病预防和恢复 |

## 6. 选拍推荐中的等级映射

| 等级 | 推荐倾向 | 不建议 |
|---|---|---|
| L0-L1 | 4U/5U，均衡或略头轻，中杆适中/偏软，容错高 | 3U 头重、特硬中杆、高磅 |
| L2 | 4U 为主，根据打法选均衡/进攻/速度 | 只看职业同款 |
| L3 | 可按打法选头重、速度、控制 | 忽视身体条件和打球项目 |
| L4-L5 | 可精细比较中杆、拍框、重量规格 | 只依据“好评最多”选择 |

# 用户打法标签体系

## 1. 说明

BWF 教练资料和公开战术分析研究说明，羽毛球可以从击球类型、落点、球员位置、移动和战术意图等角度进行分析。本文不是官方分类，而是 RAG 推荐系统使用的标签体系。所有“适合/不适合”都应与用户水平、身体条件和实际试打反馈结合。

## 2. 一级标签

| 标签 | 解释 | 常见问题 |
|---|---|---|
| attack_power | 进攻力量型 | 喜欢杀球、下压、后场主动进攻 |
| speed_drive | 速度平抽型 | 喜欢双打平抽挡、快速连贯 |
| control_placement | 控制落点型 | 重视落点、线路、节奏变化 |
| defense_counter | 防守反击型 | 接杀、防守、等待机会反击 |
| all_round | 均衡全能型 | 没有极端偏好，追求稳定 |
| net_front | 网前封控型 | 双打前场、接发、封网较多 |
| rear_court_attack | 后场进攻型 | 双打后场或单打后场下压多 |
| beginner_easy_power | 新手借力型 | 技术未成熟，希望省力、容错高 |

## 3. 与项目关联

| 项目 | 高价值标签 |
|---|---|
| 单打 | control_placement, defense_counter, attack_power, all_round |
| 双打 | speed_drive, net_front, rear_court_attack, all_round |
| 混双 | net_front, rear_court_attack, speed_drive, control_placement |

## 4. 打法标签判断依据

### 4.1 进攻力量型 attack_power

判断依据：

- 经常主动杀球或点杀；
- 更在意后场球威胁；
- 能接受一定挥速牺牲；
- 喜欢头重或中杆偏硬反馈。

选拍倾向：

- 可考虑头重或略头重；
- 中杆不宜超过自身发力能力；
- 新手不应只因想杀球而选择高难度进攻拍。

### 4.2 速度平抽型 speed_drive

判断依据：

- 双打比例高；
- 经常打平抽挡、防守反抽、接发抢快；
- 不喜欢球拍拖慢；
- 对挥速和连续性敏感。

选拍倾向：

- 4U/5U；
- 头轻或均衡；
- 中杆适中或偏硬，视力量而定。

### 4.3 控制落点型 control_placement

判断依据：

- 喜欢吊、搓、推、放、变线；
- 不只追求杀球速度；
- 重视控球感、拍面稳定和出球方向。

选拍倾向：

- 均衡或略头重；
- 中杆适中；
- 不必追求极端高磅。

### 4.4 防守反击型 defense_counter

判断依据：

- 经常被动接杀；
- 喜欢挡网、抽对角、挑后场；
- 需要快速从防守转进攻。

选拍倾向：

- 4U 或 5U；
- 头轻/均衡；
- 避免过重、过硬、过高磅组合。

### 4.5 网前封控型 net_front

判断依据：

- 双打中经常站前场；
- 接发、封网、扑球多；
- 要求拍子启动快、拍面反应快。

选拍倾向：

- 4U/5U；
- 均衡或头轻；
- 线床不宜过低磅导致控球发飘。

### 4.6 后场进攻型 rear_court_attack

判断依据：

- 双打站后场多；
- 杀球、劈杀、吊杀结合；
- 需要连续下压和后场威胁。

选拍倾向：

- 3U/4U 视力量；
- 略头重或头重；
- 中杆适中到偏硬。

## 5. 标签组合示例

```yaml
play_profile:
  primary: speed_drive
  secondary:
    - net_front
    - defense_counter
  event: doubles
  level: L2
  equipment_need:
    - 挥速快
    - 防守轻松
    - 容错高
  avoid:
    - 3U头重
    - 特硬中杆
```

## 6. 冲突处理规则

| 冲突 | 处理 |
|---|---|
| 用户说想杀球重，但手腕弱 | 优先保护身体和上手难度，推荐轻一点、别太硬 |
| 用户双打前场多，但想买极头重拍 | 解释极头重可能影响封网和防守反应 |
| 用户新手想买职业同款 | 先解释规格和难度，再给替代方案 |
| 用户膝盖/肩膀不适 | 装备推荐不能鼓励高强度打法，需提示降低训练强度 |

## DOC-13｜单打基础战术

# 单打基础战术

## 1. 战术目标

单打的核心不是每拍都杀球，而是通过落点、节奏和移动消耗对手，创造主动进攻机会。公开战术分析研究通常会记录击球类型、击球位置、双方位置等要素，这说明单打战术可以结构化为“球路 + 位置 + 意图”。

## 2. 基础战术框架

| 战术 | 目标 | 常用球路 | 风险 |
|---|---|---|---|
| 拉吊突击 | 拉开前后场，等浅球进攻 | 高远、吊球、突击杀 | 自己后场不到位会被反压 |
| 压反手 | 攻击对手反手后场弱点 | 高远到反手、劈吊、杀斜线 | 线路单一会被预判 |
| 控网抢高点 | 逼对手挑球 | 放网、搓球、推后场 | 网前质量差会被扑 |
| 防守反击 | 先稳住被动，再抓空当 | 挡网、挑后场、抽对角 | 挑球浅容易被杀 |
| 变速突击 | 打乱节奏 | 慢吊、快推、点杀 | 需要动作一致性 |

## 3. 拉吊突击

```yaml
tactic: 拉吊突击
scene: singles
objective:
  - 调动对手前后移动
  - 创造浅球或失位
key_shots:
  - 后场高远
  - 后场吊球
  - 点杀
  - 杀上网
suitable_for:
  - L2以上
  - 有基本后场能力
risk:
  - 高远球不到位会被反攻
  - 吊球过高会被扑
```

## 4. 压反手

适用条件：

- 对手反手后场被动；
- 对手转身慢；
- 对手反手回球质量低。

执行方式：

1. 先用高远或平高压到对方反手后场；
2. 对手回球浅时突击；
3. 对手站位后撤时吊网前；
4. 不要每拍都打同一区域，避免被预判。

## 5. 控网抢高点

核心逻辑：

- 通过高质量放网/搓球，迫使对手挑球；
- 对手挑球不到位时后场下压；
- 对手回放网前时抢高点扑或推。

风险：

- 网前动作太大容易暴露意图；
- 搓球过高会被直接扑死；
- 不适合基本手感不稳定的新手反复使用。

## 6. 防守反击

防守反击不是被动挨打，而是用稳定回球把对方进攻质量拉低。

常见选择：

| 对方来球 | 可选处理 |
|---|---|
| 重杀到身体 | 挡网、挡斜线、抽平 |
| 杀到边线 | 挑后场或挡网 |
| 连续下压 | 先顶后场，争取回中 |
| 对方上网慢 | 挡网前制造二次启动 |

## 7. 单打战术问诊

RAG 生成单打战术建议前，应询问或推断：

- 用户水平；
- 常见丢分方式；
- 对手类型；
- 自己后场是否到底；
- 是否步法慢；
- 体能是否够；
- 是否有伤痛。

## 8. 输出模板

```markdown
你现在的问题更像是“后场不到位导致被压制”，不是简单缺少杀球。建议先用高远球保证长度，再加入吊球变化。训练上做后场两点高远 + 吊球组合，多球每组 12-16 球，先保证回中和击球点。
```

## DOC-14｜双打基础战术


> 使用边界：本文档用于羽毛球 RAG 检索、解释和训练建议生成。涉及比赛判罚时，以 BWF 最新《Laws of Badminton》和赛事规程为准；涉及身体疼痛、伤病、康复时，仅作运动科普，不能替代医生、物理治疗师或持证教练的个体诊断。

# 双打基础战术

## 1. 双打核心

双打比单打更强调：

- 发接发前三拍；
- 抢网和封网；
- 前后/左右站位转换；
- 平抽挡速度；
- 两人分工和补位；
- 减少中路犹豫。

## 2. 基本站位

| 站位 | 场景 | 说明 |
|---|---|---|
| 前后站位 | 进攻 | 后场队员下压，前场队员封网 |
| 左右站位 | 防守 | 两人横向分区接杀 |
| 轮转站位 | 攻防转换 | 根据球路和位置动态切换 |

## 3. 进攻原则

进攻目标不是每拍杀死，而是持续压低对方回球质量。

常见方式：

- 后场下压到中路或身体；
- 前场封网不给对方起球；
- 通过点杀、劈吊、平抽保持连贯；
- 杀球后同伴准备封网，不要两人都后退。

## 4. 防守原则

双打防守站位通常更偏左右分区。防守时：

- 不要两人挤在一起；
- 中路球提前约定谁接；
- 接杀不一定只能挑球，可以挡网或抽对角；
- 防守回球高质量才能转守为攻。

## 5. 轮转原则

| 情况 | 轮转建议 |
|---|---|
| 后场队员杀球后上网 | 搭档根据空位补后场 |
| 前场队员被迫挑球 | 两人转入左右防守 |
| 对方起高球 | 一人后退进攻，一人前场封网 |
| 中路球犹豫 | 赛前约定正手优先或前场优先 |

## 6. 常见错误

| 错误 | 结果 | 修正 |
|---|---|---|
| 发球后站着不动 | 第三拍被抢 | 发球后准备封网/防推 |
| 接发只会挑后场 | 让对方直接进攻 | 加入推、扑、放、抽 |
| 两人同时抢网 | 后场空 | 明确前后分工 |
| 防守时站太窄 | 边线空 | 左右分区 |
| 后场只会重杀 | 连贯断 | 加点杀、劈吊、平抽 |

## 7. 双打战术问诊

```yaml
ask:
  - 你主要站前场还是后场？
  - 你们丢分多在发接发、平抽挡、防守接杀还是轮转？
  - 你和搭档是否固定？
  - 你更怕被推后场还是被扑网前？
```

## 8. 输出模板

```markdown
你们的问题更像是发接发后第三拍被动。先不要急着练重杀，优先练“发短球后封网”和“接发推中腰/推后场”。双打里前三拍拿不到主动，后面很容易一直防守。
```

## DOC-15｜发接发前三拍


> 使用边界：本文档用于羽毛球 RAG 检索、解释和训练建议生成。涉及比赛判罚时，以 BWF 最新《Laws of Badminton》和赛事规程为准；涉及身体疼痛、伤病、康复时，仅作运动科普，不能替代医生、物理治疗师或持证教练的个体诊断。

# 发接发前三拍

## 1. 定义

发接发前三拍通常指：

1. 第 1 拍：发球；
2. 第 2 拍：接发；
3. 第 3 拍：发球方对接发球的处理。

在双打和混双中，前三拍经常决定主动权。

## 2. 发球方目标

| 目标 | 解释 |
|---|---|
| 发球合法 | 站位、动作、落点符合规则 |
| 控制高度 | 发短球不过高，减少被扑 |
| 控制落点 | 靠近 T 点、边线或对手不舒服区域 |
| 准备第三拍 | 发球后不是结束，而是准备封网、防推、防扑 |

## 3. 接发方目标

| 接发方式 | 目的 | 风险 |
|---|---|---|
| 扑 | 直接抢攻 | 来球不够高时容易下网/触网 |
| 推 | 推到中腰或后场空当 | 推太高会被反抓 |
| 放 | 逼对方挑球 | 质量差会被扑 |
| 挑 | 摆脱被动 | 让对方获得进攻机会 |
| 抽 | 快速压制 | 需要反应和拍面稳定 |

## 4. 第三拍处理

发球方第三拍要根据接发质量选择：

| 对方接发 | 第三拍选择 |
|---|---|
| 推后场 | 后场队员快速到位，下压或过渡 |
| 放网 | 前场队员抢高点搓/推/扑 |
| 挑高 | 转入前后进攻 |
| 平抽 | 稳住拍面，争取压低回球 |

## 5. 双打练习

### 练习 1：发短球 + 封网

```yaml
drill: 发短球后封网
players: 2-4
sets: 5
reps: 10
goal:
  - 发球低
  - 发完后拍举起
  - 抢第三拍
```

### 练习 2：接发四选一

接发者轮流练：

- 扑；
- 推；
- 放；
- 挑。

目标不是只练一种“必杀”，而是根据发球质量选择。

### 练习 3：前三拍连贯

```yaml
sequence:
  - 发短球
  - 接发推中腰
  - 第三拍抽压或挡网
focus:
  - 发球后启动
  - 接发后准备下一拍
  - 两人轮转
```

## 6. 常见错误

- 发完球低头看球，不准备第三拍；
- 接发只会挑；
- 接发扑球过猛导致下网；
- 接发推球太高；
- 前场同伴不封网；
- 后场同伴没有准备补位。

## 7. RAG 输出模板

```markdown
你的双打被动大概率不是后场不够强，而是前三拍没有抢到主动。先练发短球质量和发完后的封网准备；接发不要只挑，至少加入“推中腰”和“放网”两个选择。
```

## DOC-16｜步法基础


> 使用边界：本文档用于羽毛球 RAG 检索、解释和训练建议生成。涉及比赛判罚时，以 BWF 最新《Laws of Badminton》和赛事规程为准；涉及身体疼痛、伤病、康复时，仅作运动科普，不能替代医生、物理治疗师或持证教练的个体诊断。

# 步法基础

## 0. 训练建议安全边界（2026-07-09）

- 本文档的训练计划是通用模板，不替代教练面对面纠错。
- 初学者优先建立规则、握拍、发球、稳定击球和安全移动，再追求杀球力量。
- 若用户描述疼痛、肿胀、无法负重、麻木、关节不稳等红旗信号，应优先召回《必须就医信号》，暂停训练建议。
- 成人一般体能建议可参考公共健康指南：每周累计中等强度活动约 150 分钟或高强度约 75 分钟，并结合每周至少 2 天力量训练；但羽毛球专项训练量需按个人基础递增。


## 1. 步法的重要性

羽毛球不是只靠挥拍。步法决定你能否在合适位置、合适高度、合适重心下击球。BWF 教学资料将移动能力作为基础教学内容之一。

## 2. 基础概念

| 术语 | 解释 |
|---|---|
| 启动 | 对方击球后第一时间判断并移动 |
| 回中 | 击球后回到相对合理的准备位置 |
| 跨步 | 前场或边线常用的大步接球 |
| 并步 | 保持身体稳定的连续移动 |
| 交叉步 | 后场或大范围移动中常见 |
| 垫步 | 调整节奏和距离的小步 |
| 制动 | 到位后控制身体，避免冲过头 |
| 重心 | 移动、击球和恢复的身体控制中心 |

## 3. 六点步法区域

常见训练会把场地分成：

- 左前场；
- 右前场；
- 左中场；
- 右中场；
- 左后场；
- 右后场。

训练目标不是跑得越快越好，而是：

1. 启动快；
2. 路线合理；
3. 到位稳定；
4. 击球后能回中；
5. 不因过度跨步伤膝踝。

## 4. 前场步法

前场常见移动：

- 垫步 + 跨步；
- 弓步接网前；
- 击球后蹬地回中。

注意：

- 膝盖方向尽量与脚尖一致；
- 不要膝盖内扣；
- 不要每次都大跨步硬刹；
- 网前动作要和拍法结合。

## 5. 后场步法

后场常见移动：

- 侧身转体；
- 并步或交叉步后退；
- 起跳或不跳击球；
- 击球后回中。

常见错误：

- 面向球网倒退；
- 击球点太靠后；
- 打完站在后场不回来；
- 为了杀球忽视回动。

## 6. 中场步法

中场多用于：

- 平抽挡；
- 接杀；
- 抽压；
- 过渡球。

关键：

- 重心低；
- 拍子举起；
- 小步快速调整；
- 不要大幅度后仰。

  sets: 3

    - 击球点

## 8. 安全提醒

出现膝盖、踝关节、跟腱疼痛时，应降低强度。若有明显肿胀、无法负重、关节卡住或疼痛加重，应停止训练并寻求专业帮助。

## 2. 羽毛球常见损伤部位

系统综述通常从损伤发生率、机制、部位、类型、严重程度和风险因素等维度研究羽毛球损伤。羽毛球包含快速启动、急停、变向、跳跃、上肢挥拍和重复击球，因此常见问题包括：

| 部位 | 常见问题 | 可能诱因 |
|---|---|---|

| 手腕 | 手腕疼痛、扭伤 | 被动击球、发力代偿 |
| 腰背 | 腰背酸痛 | 后仰击球、核心不足、疲劳 |
| 跟腱/小腿 | 跟腱痛、小腿拉伤 | 跳跃、蹬地、热身不足 |

## 3. 急性损伤 vs 过度使用

| 类型 | 特征 | 例子 |
|---|---|---|
| 急性损伤 | 某一次动作后突然疼痛 | 踝扭伤、膝扭伤、摔倒手腕伤 |
| 过度使用 | 逐渐出现，训练量越大越明显 | 肩肘痛、髌腱痛、跟腱痛、腰背痛 |

## 4. 初步处理原则

对于轻微软组织不适，可考虑：

- 停止诱发动作；
- 降低训练强度；
- 休息和观察；
- 冰敷急性肿痛部位；
- 避免带痛比赛；
- 复盘动作和装备。

注意：不要给用户开药、诊断病名或承诺恢复时间。

## 5. 装备相关风险

| 装备因素 | 可能影响 |
|---|---|
| 鞋子防滑差 | 增加滑倒和扭伤风险 |
| 鞋子支撑不足 | 踝膝稳定性下降 |

## 2. 赛前热身结构

| 阶段 | 内容 | 时间 |
|---|---|---|
| 升温 | 慢跑、跳绳轻量、侧滑步 | 3-5 分钟 |

| 专项挥拍 | 正手、反手、高远、网前轻挥 | 3-5 分钟 |

| 动作 | 目标 |
|---|---|
| 肩绕环 | 肩关节准备 |

| 小碎步启动 | 反应和脚踝准备 |
| 侧向滑步 | 双打防守准备 |
| 后退侧身步 | 后场击球准备 |

| 部位 | 放松方式 |
|---|---|
| 小腿 | 靠墙小腿拉伸 |
| 股四头肌 | 站姿大腿前侧拉伸 |
| 腘绳肌 | 大腿后侧拉伸 |
| 臀部 | 坐姿臀部拉伸 |
| 前臂 | 手腕屈伸肌群拉伸 |
| 肩背 | 胸肩打开、背阔肌放松 |

## 5. 不建议

- 不热身直接比赛；
- 静态拉伸后立刻进行最大强度启动；
- 疼痛部位强行拉伸；
- 用拉伸替代休息和治疗；
- 肌肉明显拉伤后继续运动。

```

如果热身后疼痛没有缓解，或出现肿胀、无法负重、关节不稳等情况，应停止运动并寻求专业评估。

## 2. 运动损伤红旗信号

出现以下情况，应建议停止运动并尽快就医或寻求急诊/专业医疗帮助：

| 信号 | 说明 |
|---|---|
| 无法负重 | 例如脚踝、膝盖受伤后不能走路或不能正常踩地 |
| 明显肿胀或迅速肿胀 | 特别是伴随疼痛、发热或活动受限 |
| 明显畸形 | 怀疑骨折、脱位等情况 |
| 剧烈疼痛 | 超出普通酸痛，影响正常活动 |
| 关节不稳或打软腿 | 膝踝等关节无法稳定支撑 |
| 关节卡住/锁住 | 膝关节不能正常伸屈 |
| 麻木、刺痛、感觉异常 | 可能涉及神经或循环问题 |
| 受伤时听到/感觉“啪”或“爆裂” | 需警惕韧带、肌腱或骨损伤 |
| 发热、红肿热痛 | 可能存在感染或炎症风险 |
| 疼痛持续加重 | 休息后仍恶化 |
| 反复同一部位受伤 | 需专业评估原因 |

## 3. 不同部位示例

### 踝关节

应就医/评估：

- 不能负重；
- 肿胀和疼痛数日不缓解或加重；
- 反复扭伤；
- 明显不稳。

### 膝关节

应就医/评估：

- 膝盖锁住或打软；
- 热、肿、明显压痛；
- 急性扭伤后迅速肿胀；
- 不能继续活动或不能负重。

### 肩肘腕

应就医/评估：

- 明显无力；
- 夜间痛；
- 麻木刺痛；
- 活动范围明显下降；
- 疼痛持续不缓解。

### 用户问：“疼还能打吗？”

```
