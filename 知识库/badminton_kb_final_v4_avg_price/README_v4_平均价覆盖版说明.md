# BadmintonGear-AI 知识库 v4：平均价覆盖版

生成日期：2026-07-10

## 本次处理

按你的要求，对 v3 中价格未对齐/价格不同的记录重新处理：

1. 对有知识库对应记录、但清单价与知识库参考价不一致的行，按以下规则重新计算：
   - 清单侧：同一知识库 record 下的价格不一致清单行取均值；
   - 知识库侧：原知识库参考价取均值，价格范围按范围内数值取均值；
   - 最终覆盖价 = `(清单均值 + 知识库均值) / 2`，四舍五入到元。
2. 将最终覆盖价同时写回：
   - 商品清单标准化文件；
   - 结构化知识库 JSONL；
   - RAG 检索 Markdown。
3. 对原本完全未对应知识库的 `川崎 入门均衡拍`，本版新增最小知识库记录，价格沿用清单参考价。
4. 价格仍统一为参考价，非实时成交价；本系统不做交易、库存或下单判断。
5. 输出文件中已移除资料占位用语。

| 文件 | 用途 |
|---|---|
| `product/商品库球拍标准化_v4_平均价覆盖版.csv` | 后台商品清单，可直接替换旧商品标准化表 |

| `product/羽毛球拍商品库_RAG检索_v4_平均价覆盖版.md` | 向量库入库文本，一支球拍一个 chunk |

| `reports/知识库记录平均价覆盖明细.csv` | 知识库 record 级覆盖明细 |

| `reports/清单新增到知识库记录.csv` | 原未对应但本版新增到知识库的记录 |

## 注意

本版按你的要求主动覆盖价格。对于某些本来属于“模糊对应/低置信对应/泛型号”的记录，价格已经统一，但名称与型号对应关系仍建议后续人工确认，避免把不同版本或不同系列的球拍长期绑定到同一 record。
